# LLM-Evaluator-FastAPI
FastAPI-based web application that combines an e-commerce support chatbot with multi-model LLM benchmarking.  Users can chat with different LLM providers (Groq, Gemini, Mistral, OpenRouter), rate responses, and view daily/weekly leaderboards.  Built with FastAPI (async), PostgreSQL, SQLAlchemy, JWT authentication, and Cookies.
